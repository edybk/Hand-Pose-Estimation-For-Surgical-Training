# Training on the Open Surgery Simulation Dataset with MMDetection

- Install MMDetection and download the dataset:
    ```
    ./setup.sh
    ```
- Train the YOLO-X object detection model
    ```
    ./train.sh
    ```
- Test the trained model
    ```
    ./test.sh
    ```
- Visualize the results
    ```
    ./visualize.sh
    ```
