# Training on the Open Surgery Simulation Dataset with MMPose

- Install MMPose and download the dataset:
    ```
    ./setup.sh
    ```
- Train the Simple Baseline Resnet pose estimation model
    ```
    ./train.sh
    ```
- Test the trained model
    ```
    ./test.sh
    ```
- Visualize the results
    ```
    ./visualize.sh
    ```
